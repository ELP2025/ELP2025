# ELP2025 Projet Elm : TCTurtle
Dans ce projet, nous allons développer une version simplifié de Turtle en Elm

# Compilation :
## Version "production"
`elm make src/Main.elm`

## Version "test"
`elm make src/Test.elm`

# Utilisation
En cas de problème, utiliser un serveur http pour accéder à la page peut régler les problèmes.
On peut ensuite entrer des programmes Turtle dans le champ de texte prévu à cet effet. La syntaxe des programmes est défini dans le cahier des charges du projet.
